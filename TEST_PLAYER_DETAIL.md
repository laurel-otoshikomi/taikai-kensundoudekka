# 選手詳細表示機能 - テストガイド

## 📋 機能概要

大会順位ページで、各選手の行をタップすると、その選手の詳細情報と全釣果記録を表示するモーダルが開きます。

## ✨ 実装内容

### 1. ランキング行にクリックイベントを追加
- 各ランキング行に `onclick="showPlayerDetail(ゼッケン番号)"` を設定
- カーソルをポインターに変更（`cursor: pointer`）
- 視覚的なヒント「👆 タップで詳細」を表示

### 2. 選手詳細モーダル
モーダルに表示される情報：
- **基本情報**: ゼッケン番号、選手名、所属クラブ
- **統計情報**:
  - 総釣果数（total_count）
  - 最大長寸（max_len）
  - 最小長寸（min_len）
  - 最大重量（max_weight）
  - 最小重量（min_weight）
  - 総重量（total_weight）
  - リミット長寸合計（limit_total_len）
  - リミット重量合計（limit_weight）
- **全釣果記録**: 各釣果の長寸・重量・登録日時をリスト表示

### 3. モーダルの特徴
- スマホ対応のフルスクリーンレイアウト
- スムーズなフェードイン/アウトアニメーション
- 背景タップまたは×ボタンで閉じる
- スクロール可能な釣果リスト
- カスタムスクロールバー
